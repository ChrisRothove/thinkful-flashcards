# Flashcard app created for Thinkful assignment

## Skills used:

- Javascript
- JSX/React (+hooks)
- Bootstrap 4
- HTML5

## Basic Function:

- users can create decks of flashcards and edit/study their contents
- uses JsonAPI to simulate API calls to a built-in Json database
